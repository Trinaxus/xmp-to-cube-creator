# xmp to cube creator

Ein Web-Tool, um Adobe XMP-Presets in **.cube LUTs** zu konvertieren. 

- XMP laden
- Referenzbild wählen
- Farbkorrekturen / Color Grading anpassen
- LUT-Größe und Farbraum wählen
- .cube-Dateien exportieren

## Entwicklung

Voraussetzung: Node.js und npm.

```bash
npm install
npm run dev
```

Build für Produktion:

```bash
npm run build
```

Technologien:

- Vite
- React + TypeScript
- Tailwind CSS
- shadcn/ui
